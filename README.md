# 🍬 Nassau Candy Distributor — Factory Reallocation Optimizer

A machine learning-powered web app that predicts shipping lead times 
and recommends optimal factory reassignments.

- ML Model Performance
- Factory Simulator
- Recommendations Dashboard
- Risk & Impact Panel

## 🛠️ Built With
