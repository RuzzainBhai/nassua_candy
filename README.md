# nassua_candy
# 🍬 Nassau Candy Distributor — Factory Reallocation Optimizer

A machine learning-powered web app that predicts shipping lead times 
and recommends optimal factory reassignments for Nassau Candy Distributor.

## 🚀 Live App
👉 [Click here to open the app](https://b7pt3cebfwrklchgahnu8x.streamlit.app)

## 📊 Features
- EDA Overview — Lead time, profit margin, shipping analysis
- ML Model Performance — Linear Regression, Random Forest, Gradient Boosting
- Factory Simulator — Predict lead time across all 5 factories
- Recommendations — Ranked factory reassignment suggestions
- Risk & Impact Panel — Profit impact and route cluster analysis

## 🗂️ Files
- `nassau_candy_app.py` — Main Streamlit application
- `nassau_candy_distributor.csv` — Dataset (10,194 orders)
- `requirements.txt` — Python dependencies

## 🛠️ Built With
- Python, Streamlit, Pandas, Scikit-learn, Plotly
